# Paradise Nursery - Shopping Cart Application

## Project Overview
Paradise Nursery is a dynamic e-commerce web application for an online plant shop. 
It allows users to browse plants, view details, and manage a shopping cart 
with real-time updates.

## Features
- 🌿 Browse a wide variety of plants with images, names, descriptions, and prices
- 🛒 Add plants to a shopping cart
- ➕➖ Adjust item quantities directly in the cart
- ❌ Remove items from the cart
- 💰 Real-time total cost calculation
- 🔢 Dynamic cart quantity updates in the navigation bar
- 📱 User-friendly and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript (React / or whatever framework you used)

## How to Run the Project
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the application: `npm start`
4. Open your browser at `http://localhost:3000`

## Project Purpose
This project was built as a final project to apply skills in building 
interactive, functional e-commerce web applications.
